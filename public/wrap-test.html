intentmap.io wrap — isolation test page

This page exists for one purpose: validate that the wrapped DataMoon pixel operates end-to-end with zero direct vendor-host calls in the browser's network panel.

No baseline pixels are loaded here, so the wrapped pixel has clean access to window.liQHub and can run uncontested.

Expected network activity from the pixel:

  • cdn.intentmap.io/v1/track.min.js
  • li.intentmap.io/did-00a6.min.js
  • li.intentmap.io/i/sync-container, li.intentmap.io/idx/idex, li.intentmap.io/rp/...
  • t.intentmap.io/api/v1?md5=... (after LI resolves an md5)

Expected to not appear:

  • Anything on *.liadm.com
  • Anything on *.datamoon.com or *.retargetiq.com
